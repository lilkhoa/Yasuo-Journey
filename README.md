# Yasuo Game

Yasuo Game is a Mario-like platformer game featuring Yasuo from League of Legends. The game is built using Python and the Pygame library, offering an engaging 2D platforming experience with unique mechanics inspired by the League of Legends universe.

## Features

- **Playable Yasuo Character**: Control Yasuo with his signature abilities, including Q, W, and E skills.
- **Dynamic Combat System**: Skills scale with Yasuo's attack damage and level, providing a strategic gameplay experience.
- **Interactive Environment**: Push boxes and barrels, interact with chests, and navigate through a variety of obstacles.
- **NPCs and Bosses**: Battle against Ghosts, Shooters, Onre, and powerful bosses with unique abilities.
- **Skill System**: Upgrade Yasuo's skills to increase their damage and reduce cooldowns.
- **Checkpoints**: Save your progress and respawn at the last checkpoint upon death.
- **Game Over and Victory Screens**: Experience a polished game flow with game over and victory sequences.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd A3_Yasuo
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
4. Run the game:
   ```
   python main.py
   ```

## Gameplay

- **Movement**: Use the arrow keys to move Yasuo left or right.
- **Jumping**: Press the spacebar to jump.
- **Skills**:
  - **Q**: Perform a quick thrust attack.
  - **W**: Summon a wind wall to block projectiles.
  - **E**: Dash through enemies or obstacles.
- **Interacting with Objects**:
  - Push boxes and barrels to solve puzzles or clear paths.
  - Break barrels to collect items.
  - Open chests to find rewards.
- **Checkpoints**: Activate checkpoints to save your progress.
- **Objective**: Defeat enemies, avoid obstacles, and reach the end of the level to win.
